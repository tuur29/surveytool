# Colors

The colors object contains multiple types of colors, most elements have a background and foreground (`on`) color.
The main 4 color sets are:

| Background Color                        | Foreground Color                          | Description                        |
| --------------------------------------- | ----------------------------------------- | ---------------------------------- |
| <ColorText color="primary" lightText /> | <ColorText color="onPrimary" />           | Main accent color                  |
| <ColorText color="back" />              | <ColorText color="onBack" lightText />    | Base background / foreground color |
| <ColorText color="surface" />           | <ColorText color="onSurface" lightText /> | Colors for raised elements (cards) |
| <ColorText color="error" />             | <ColorText color="onError" />             | Used for displaying errors         |

In addition there are also colors that are not paired up:

-   <ColorText color="spinner" lightText />: accent color of the loading indicator
-   <ColorText color="shadow" lightText />: color of text and box (surface elevation) shadows
-   <ColorText color="icon" /> & <ColorText color="iconHover" lightText />: info icons
-   <ColorText color="separator" />: color of lines and dividers between elements

## Control Colors

The biggest part of the colors object consists of theming the `controls` (as in form controls / inputs).  
These are: buttons, text fields, dropdowns, sliders...

| Type                | Colors                                                                                                                                                                                                                           | Note                                                          |
| ------------------- | -------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- | ------------------------------------------------------------- |
| Background & Border | <ColorText color="controlBack" /><br /><ColorText color="controlHighlight" lightText /><br /><ColorText color="controlBorder" /><br /><ColorText color="controlBackDisabled" /><br /><ColorText color="controlOnBackDisabled" /> | Default colors without user input                             |
| Text                | <ColorText color="controlPlaceholder" />                                                                                                                                                                                         | Most text colors are defined by the default foreground color. |
| Hover states        | <ColorText color="controlBackHover" /><br /><ColorText color="controlBorderHover" lightText />                                                                                                                                   |                                                               |
| Active states       | <ColorText color="controlHighlightActive" /><br /><ColorText color="controlBorderActive" />                                                                                                                                      | Color of selected items, checkmark and button hover           |
| Buttons             | <ColorText color="controlButton" lightText /><br /><ColorText color="controlOnButton" /><br /><ColorText color="controlButtonDisabled" /><br /><ColorText color="controlOnButtonDisabled" lightText />                           |                                                               |
| Slider              | <ColorText color="controlTick" /><br /><ColorText color="controlSliderBack" /><br /><ColorText color="controlSliderDisabled" /><br /><ColorText color="controlSliderOnDisabled" />                                               |                                                               |

## Color Palettes

Each default theme is based on 3 palettes:

**Primary**

Used as main accent color.  
They are based on the [Material Design](https://material.io/design/color/the-color-system.html#color-theme-creation) "Red" color.

<div>
    {Object.values(palettes.primary).map((value) => (
        <ColorSwatch value={value} size={30} />
    ))}
</div>

**Secondary**

Extra accents to contrast with the primary color (warnings for example).  
They are based on the [Material Design](https://material.io/design/color/the-color-system.html#color-theme-creation) "Orange" color.

<div>
    {Object.values(palettes.secondary).map((value) => (
        <ColorSwatch value={value} size={30} />
    ))}
</div>

**Layout**

These grey values are used for text, backgrounds, shadows...  
They represent 8 stops between black and white at these percentages: 0%, 3%, 6%, 9%, 15%, 21%, 36%, 57%, 93%, 100%

Light mode

<div>
    {Object.values(palettes.neutralLight).map((value) => (
        <ColorSwatch value={value} size={30} />
    ))}
</div>

Dark mode

<div>
    {Object.values(palettes.neutralDark).map((value) => (
        <ColorSwatch value={value} size={30} />
    ))}
</div>
