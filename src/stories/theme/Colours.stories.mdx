# Colours

The colors object contains multiple types of colors, most elements have a background and foreground (`on`) color.
The main 4 color sets are:

| Background Color | Foreground Color | Description |
|---|---|---|
| `primary` <ColourSwatch name="Primary" value={colors.primary} size={16} /> | `onPrimary` | Main accent color |
| `back` <ColourSwatch name="Back" value={colors.back} size={16} /> | `onBack` | Base background / foreground color |
| `surface` <ColourSwatch name="Surface" value={colors.surface} size={16} /> | `onSurface` | Colors for raised elements (cards) |
| `error` <ColourSwatch name="Error" value={colors.error} size={16} /> | `onError` | Used for displaying errors |

In addition there are also colors that are not paired up:

- `spinner`: accent color of the loading indicator
- `shadow`: color of text and box (surface elevation) shadows
- `icon` / `iconHover`: info icons
- `separator`: color of lines and dividers between elements

## Control Colors

The biggest part of the colors object consists of theming the `controls` (as in form controls / inputs).  
These are: buttons, text fields, dropdowns, sliders...

| Type | Colors | Note |
|---|---|---|
| Background & Border | `controlBack`, `controlHighlight`, `controlBorder` | Default colors without user input |
| Text | `controlPlaceholder` | Most text colors are defined by the default foreground color. |
| Hover states | `controlBackHover`, `controlBorderHover` |  |
| Active states | `controlHighlightActive`, `controlBorderActive` | Color of selected items, checkmark and button hover |
| Buttons | `controlButton`, `controlOnButton`, `controlButtonDisabled`, `controlOnButtonDisabled` | |
| Slider | `controlTick`, `controlSliderBack` | |

## Message Colors

| Type| Background | Foreground | Border |
|---|---|---|---|
| Error | `messageError` | `onMessageError` | `messageErrorBorder` |
| Info | `messageInfo` | `onMessageInfo` | `messageInfoBorder` |
| Neutral | `messageNeutral` | `onMessageNeutral` | `messageNeutralBorder` |

## Color Palettes

Each default theme is based on 3 palettes:

**Primary**

Used as main accent color.  
They are based on the [Material Design](https://material.io/design/color/the-color-system.html#color-theme-creation) "Red" color.

<div>
    {Object.values(palettes.primary).map((value) => (
        <ColourSwatch value={value} size={30} />
    ))}
</div>

**Secondary**

Extra accents to contrast with the primary color (warnings for example).  
They are based on the [Material Design](https://material.io/design/color/the-color-system.html#color-theme-creation) "Orange" color.

<div>
    {Object.values(palettes.secondary).map((value) => (
        <ColourSwatch value={value} size={30} />
    ))}
</div>

**Layout**

These grey values are used for text, backgrounds, shadows...  
They represent 8 stops between black and white at these percentages: 0%, 3%, 6%, 9%, 15%, 21%, 36%, 57%, 93%, 100%

Light mode
<div>
    {Object.values(palettes.neutralLight).map((value) => (
        <ColourSwatch value={value} size={30} />
    ))}
</div>

Dark mode
<div>
    {Object.values(palettes.neutralDark).map((value) => (
        <ColourSwatch value={value} size={30} />
    ))}
</div>
